bauer
=====

**Bayesian Estimation of Perceptual, Numerical and Risky Judgements.**

*bauer* is a PyMC-based framework for fitting hierarchical Bayesian cognitive models
to behavioural decision-making data (psychophysics, magnitude comparison, risky choice).
